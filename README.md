# Transistor-Level Neuron Design (ECE 555 Final Project)

This repository contains the final project for **ECE 555 – Digital Circuits and Components**, focused on designing and analyzing a **transistor-level neuron circuit** composed of CMOS logic gates. The project explores how digital logic and circuit-level design can emulate basic neural computation.

## 🧠 Project Overview

The goal of this project was to design a **neuron-like digital circuit at the transistor level**, using CMOS gate primitives to implement weighted summation and thresholding behavior. The design was implemented and simulated using **Cadence Virtuoso** on Linux-based systems.

Key objectives:
- Design CMOS-based logic gates at the transistor level  
- Construct a neuron architecture using digital circuit blocks  
- Verify functionality through circuit simulation  
- Analyze performance metrics such as delay, power, and noise margins  

## ⚙️ Tools and Technologies

- **EDA Tool:** Cadence Virtuoso  
- **Platform:** Linux  
- **Technology:** CMOS transistor-level design  
- **Simulation:** cscope
- **Design Components:** NAND, NOR, XOR, inverters, multiplexers, custom combinational logic  

## 🔬 Design Description

The neuron circuit is composed of:
- Transistor-level CMOS gates for signal processing  
- Logic blocks to emulate weighted input aggregation  
- Thresholding logic to produce a binary neuron output  

The design demonstrates how digital circuits can approximate neural computation at the hardware level, highlighting trade-offs between **area, power, and latency**.

## 🧪 Simulation and Verification

- Functional verification performed using Cadence simulation tools  
- Timing and power characteristics analyzed for different input patterns  
- Noise margin and stability evaluated at the transistor level  

## 📈 Results and Analysis

Key results include:
- Correct logical neuron behavior verified through waveform simulations  
- Measured propagation delay and power consumption across gate blocks  
- Discussion of scalability and potential extensions to neuromorphic hardware
